## 使い方

PCを想定しておりますが、レスポンシブ対応もしておりますのでスマホからもご利用いただけます。

## 機能一覧

- ユーザー登録、ログイン機能 (devise)
- ゲストログイン機能
- 支出登録機能
- 収入登録機能
- 月の支出、収入をグラフで可視化 (chartkick)
- 全体の支出、収入をグラフで可視化 (chartkick)
- 検索機能

## テスト方法

- RSpecを使用して以下の領域をテスト:
  - モデル
  - システム
  - リクエスト

## デプロイ方法

- Herokuを使用してデプロイします。
